¿Qué es Open Library?

Open Library es un proyecto de la Internet Archive que ofrece acceso a millones de libros disponibles en línea de forma gratuita. Es una iniciativa para crear una página web sobre cada libro jamás publicado.

¿Qué puedes buscar? Por título del libro Por autor Por género o categoría Por ISBN Por año de publicación Características principales Base de datos con millones de libros Información detallada de cada publicación Acceso libre y gratuito API pública para desarrolladores
